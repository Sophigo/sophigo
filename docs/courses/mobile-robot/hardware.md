# 底盘控制链与微处理器集成

移动机器人的稳定移动高度依赖底层执行机构（电机、编码器）与控制电路之间的精确配合。本章节详细介绍控制器控制链的硬件搭建。

---

## 1. 电机闭环调速控制链

控制小车速度的核心在于**增量式 PID 闭环算法**，将实测轮转速调节至目标转速。

```
  +---------+   Target Velocity   +-------------+  PWM Control  +--------------+
  |  ROS 2  | ------------------> | MCU / Duo   | ------------> | Motor Driver |
  | (上位机) | <------------------ | (底盘控制器) | <------------ |   (H-Bridge) |
  +---------+   Odom / feedback   +-------------+ Encoder pulse +--------------+
                                         ^
                                         |
                                         v
                                  +-------------+
                                  |  PID Loop   |
                                  +-------------+
```

### 1.1 增量式 PID 算法公式
在底盘微控制器中，每个控制周期（通常为 10ms-20ms）都会计算一次 PWM 占空比输出量：

$$
\Delta u(k) = K_p [e(k) - e(k-1)] + K_i e(k) + K_d [e(k) - 2e(k-1) + e(k-2)]
$$

$$
u(k) = u(k-1) + \Delta u(k)
$$

* $e(k)$: 目标脉冲计数与当前周期霍尔编码器实际反馈脉冲计数的差值（速度误差）。
* $u(k)$: 输出到 H 桥芯片的 PWM 占空比信号。

## 2. 算能 SophiGo Duo 控制核心引脚映射

SophiGo Duo 开发板（基于 C906 RISC-V 架构）在我们的底盘（Sophicar）方案中担任核心底盘控制板，通过多路硬定时器捕获编码器脉冲，并生成 PWM 驱动信号。

### 2.1 外设引脚定义示例

| 引脚编号 | 主控引脚分配 | 硬件外设功能 | 连接元器件说明 |
| :--- | :--- | :--- | :--- |
| **Pin 1** | `PWM_A1` | PWM 硬件输出 | 左电机前行 H 桥通道 A-IN1 |

## 3. 电机驱动板硬件设计要点
1. **防光耦隔离**: 强电（电机动力电池，如 12V）与弱电控制系统（Duo，5V/3.3V）通过光耦隔离，防止电机反向电动势击穿芯片。
2. **电流检测**: 在 H 桥下桥臂接入毫欧级采样电阻（Shunt Resistor），利用 Duo 的 ADC 引脚监测电机运行电流，防止电机堵转烧毁。
3. **稳压系统**: 12V 动力电池经由 DC-DC Buck 降压芯片（如 MP2307 或 LM2596）稳压至 5V 供给上位机和算能 Duo 主板。

## 4. 实验课：电机 PID 控制参数校准
1. **P 参数调节**: 从零开始调大比例系数 $K_p$，直到轮子在给定速度时发生轻微振动，再乘以 $0.6$ 回退。
2. **I 参数调节**: 增加积分系数 $K_i$，用于消除系统稳态误差，缩短静差调节时间。
3. **D 参数调节**: 视载荷和摩擦力情况微调微分系数 $K_d$，用于抑制车身瞬间加速时的超调。
