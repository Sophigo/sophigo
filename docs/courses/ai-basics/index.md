# AI 应用基础课程

本课程旨在帮助学习者掌握如何将现代人工智能（LLM、视觉模型及生成式 AI）应用到工科研发与日常创新流程中。

## 课程概述

AI 正在重构我们与机器、代码和设计的交互方式。本课程不仅介绍 AI 的基本理论，更侧重于工程实践中的具体落地应用。

### 核心板块

1. **大语言模型与 Prompt 工程**
   - 掌握结构化 Prompt 的设计原则。
   - 在硬件选型、代码编写和技术文档撰写中高效使用 AI 助手。

2. **生成式 AI 与工业设计**
   - 使用 Stable Diffusion、Midjourney 等生成工具辅助产品外观与 CMF 设计。
   - 从文本概念快速转化为高质量视觉效果图。

3. **智能体 (Agents) 开发入门**
   - 理解 Agent 架构体系（感知、规划、记忆与执行）。
   - 编写简易的硬件部署/测试辅助 Agent。

---

## 实践研讨与划词共创

本课程的所有学习材料均支持 **Hypothesis 划词批注** 讨论。请在阅读过程中高亮关键句，并与社群里的其他创客共同探讨 AI 落地应用的最佳实践。
