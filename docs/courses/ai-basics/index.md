## 📌 1. 课程定位

**时代之变（Why）：** 2023 年以来，以 ChatGPT 为代表的大语言模型和各类 AIGC 工具正以前所未有的速度渗透到各行各业。无论是内容创作、数据分析还是产品开发，AI 已不再是一个可选项，而是职场核心竞争力的一部分。未来的高效工作者，核心壁垒不再是单一的操作技能，而是**定义需求、驾驭 AI 工具进行系统性决策**的能力。

**方法之新（How）：** 本课程采用 **"需求驱动 + 实战导向"** 的教学模式，紧紧围绕真实工作场景展开。通过"需求拆解 → AI 工具实操 → 产品落地"的完整链路，帮助参训人员在 3 天高密度集训中快速掌握 AI 工具的应用方法论，并输出可直接投入使用的 AI 原生工具/产品原型。

**成果之硬（What）：** 课程涵盖 AI 需求分析、Prompt 工程、AIGC 全工具链实操、电商数据抓取分析、数字人开发、Web 可视化等六大模块。每位学员将在导师指导下完成一个完整的 AI 原生产品原型，并参与最终路演与结业评审。

### 1.1 核心能力培养

在整个学习过程中，参与者将共同解决实际问题，从而培养以下核心能力：

- **AI 需求分析与拆解**：精准描述业务痛点，将模糊需求拆解为可执行的 MVP。
- **Prompt 工程**：掌握目标-要求-约束三要素向 AI 提出需求的技巧，获取高质量输出。
- **AIGC 全工具实操**：覆盖图文、PPT、视频生成等场景的国内外主流 AI 工具运用。
- **数据驱动决策**：利用可视化爬取 + AI 分析实现数据采集、清洗与报告生成。
- **数字人开发基础**：理解 RAG 原理，掌握 2D 数字人交互实现流程。
- **AI 原生产品设计**：从需求到原型，独立输出 AI 原生工具或产品方案。

## 📋 2. 课程大纲 — 三日日程安排

> **课程目标：** 本次培训聚焦 AI 工具基础应用与实际工作场景落地，帮助参训人员将 AI 技术应用于 IP 形象打造、数字人开发、电商数据抓取分析等工作场景，最终能独立输出 AI 原生产品方案。

### DAY 1 — AI 需求拆解与 AIGC 实操

| 时段 | 模块 | 核心内容 |
|---|---|---|
| **上午** | 模块 1：AI 需求拆解与 MVP 落地思维 | • 解析 AI 与工作的融合逻辑，明确 AI 使用边界与核心价值<br>• 教授 AI 需求分析方法，实现从主观判断到用户真实需求的转变<br>• 讲解模糊痛点拆解为可执行 MVP 的核心步骤与方法<br>• 结合场景梳理 AI 应用框架，完成《AI 产品说明书》初稿 |
| **下午** | 模块 2：AIGC 全工具实操与场景应用 | • 讲解 Prompt 核心撰写技巧，掌握不同场景的设计逻辑<br>• 介绍国内外主流 AIGC 工具，覆盖图文、PPT、简易视频生成<br>• 演示各类 AIGC 工具搭配策略，适配创意工作不同需求<br>• 实操指导学员用工具完成 IP 形象设计与报告生成 |

### DAY 2 — 数据抓取分析与数字人

| 时段 | 模块 | 核心内容 |
|---|---|---|
| **上午** | 模块 3：电商数据采集分析与 Web 可视化 | • 教学可视化爬取技术，实现多平台电商数据高效抓取<br>• 讲解 Cursor+NotebookLM 用法，实现对话式数据清洗与分析<br>• 教授 Prompt 定制技巧，实现秒级报告与图文一键排版<br>• 借助 Google AI Studio，实现分析数据的 Web 开发与可视化 |
| **下午** | 模块 4：数字人开发与 API 交互实操 | • 讲解 RAG、NotebookLM 核心原理，夯实数字人开发基础<br>• 介绍 Web 前后端基础，解析数字人 API 调用逻辑与方法<br>• 演示 2D 数字人简易交互实现流程，开展现场实操指导<br>• 结合实际创意场景，梳理数字人开发核心环节与优化方向 |

### DAY 3 — 项目卡点攻坚与路演结业

| 时段 | 模块 | 核心内容 |
|---|---|---|
| **上午** | 模块 5：AI 项目开发卡点攻坚与推进 | • 收集学员实操问题，由导师进行集中解答<br>• 针对学员项目进度，开展一对一推进指导<br>• 帮助定位项目关键卡点，提供落地解决思路<br>• 督促梳理项目当前进展，提炼核心问题并提交 |
| **下午** | 模块 6：AI 原生产品路演 & 结业仪式 | • 学员开展产品路演，展示自主开发的 AI 原生工具/产品原型<br>• 导师从产品逻辑、技术实现、用户体验维度进行专业点评与结业指导<br>• 举行结业仪式，颁发培训结业证书 |

## 🏆 4. 课程产出与评估

### 4.1 核心交付物

- **《AI 产品说明书》**：针对实际工作场景的 AI 应用需求分析文档。
- **IP 形象设计作品**：使用 AIGC 工具完成的企业或个人 IP 形象。
- **电商数据分析报告**：从数据抓取到可视化呈现的完整分析报告。
- **数字人交互原型**：2D 数字人简易交互演示。
- **AI 原生产品原型**：包含完整的方案文档与可演示的产品原型，参与最终路演。

### 4.2 评估维度

| 评估维度 | 权重 | 标准 |
|---|---|---|
| AI 工具使用能力 | 30% | 工具选择的合理性、Prompt 质量、AI 输出成果的创新性 |
| 项目完成度 | 35% | 产品原型的完整度、功能实现质量、路演表现 |
| 场景落地价值 | 25% | 解决真实痛点的有效性、方案的可实施性 |
| 学习参与度 | 10% | 课堂互动、团队协作、反馈迭代 |
