# 移动机器人软硬件集成课程

本课程带你深入浅出地掌握现代移动机器人（Mobile Robots）的软硬件架构设计。我们将从最底层的电机控制、运动学建模，一直贯通到高层的 ROS2 导航与三维感知系统。

## 1. 机器人运动学模型

在主流的移动机器人设计中，我们经常使用双轮差速（Differential Drive）和麦克纳姆轮（Mecanum Wheels）底盘：
- **双轮差速底盘:** 结构简单，成本低廉，适用于室内平坦地面及扫地机器人等场景。
- **麦克纳姆轮底盘:** 具备全向平移能力，能在狭小空间内灵活转向。

> [!IMPORTANT]
> 无论是何种构型，核心在于建立轮轴转速与机器人整体线速度、角速度之间的映射矩阵。
> 掌握**移动机器人底盘的控制算法**是确保机器人路径跟踪精度（如 Pure Pursuit 或 MPC 算法）的关键。

## 2. 硬件控制链与 Sophicar 仿真

在我们的实验平台中，推荐使用基于 ARM / RISC-V 的微控制器（如 STM32 或 Sophigo Duo）进行底盘运动控制。
1. **电机驱动:** 使用 H 桥驱动芯片，配合增量式霍尔编码器，实现速度 PID 闭环控制。
2. **三维底盘仿真:** 开发者可以利用 Sophicar 仿真组件，在 Web 浏览器内直接渲染 3D 机器人车身，验证控制脉冲。
