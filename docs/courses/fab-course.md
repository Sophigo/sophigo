# Fab 数字化快速原型课程

数字化快速原型技术（Rapid Prototyping）是将 AI 生成的硬件创意具象化为物理实体的桥梁。本课程带您走进 Fab Lab (数字化制造实验室)，熟练掌握各类微型数字制造设备的安全操作与设计规范。

## 1. 3D 打印工艺与材料 (3D Printing)

3D 打印（增材制造）是快速验证外观结构与组装公差的最佳手段。
- **FDM 熔融沉积:** 常用材料为 PLA、PETG 及 ABS。打印外壳时建议壁厚不小于 1.6mm。
- **SLA 光固化:** 适用于高精度机械结构与按钮件，表面质量极高但强度相对较脆。

## 2. 激光切割与雕刻 (Laser Cutting)

激光切割（减材制造）可快速加工薄板件，如压克力、椴木板以及薄皮套。
- **排版规范:** 红色线条表示切割（Cut），黑色填充表示扫描雕刻（Engrave）。
- **缝隙补偿 (Kerf):** 切割时激光束具有一定宽度（通常为 0.1mm - 0.2mm），设计榫卯拼接时需考虑过盈配合。
