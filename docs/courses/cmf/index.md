## 📌 1. CMF 设计定位

**时代之变（Why）：** 在数字化制造时代，原型的外观（Color）、质感（Material）与工艺（Finishing）已不再是产品开发链末端的装饰环节，而是从概念阶段就应当深度介入的工程决策维度。一个优秀的 CMF 设计，能将功能结构、用户体验与制造经济性统一在同一视觉语言之下。

**方法之新（How）：** 本卡片库以 ProFabX Prototype 的知识体系为骨架，将常见的原型制造材料、增材/减材工艺、表面处理方法与工作流规范整理为标准化的知识卡片。通过 **"材料—工艺—后处理"** 的三层分类体系，帮助学习者在面对具体原型需求时，能够快速做出最优的工艺链决策。

**成果之硬（What）：** CMF 卡片库覆盖 **13 张核心知识卡片**，横跨 **6 大工艺域**（材料、增材、减材、成型、表面、工作流），每张卡片均包含材料选型、执行路径、输出目标及外部参考链接。

### 1.1 核心能力培养

通过 CMF 卡片库的学习与实践，培养以下核心能力：

- **材料选择决策**：在工程塑料、金属、复合材料之间，根据刚性/韧性/耐温/成本做出科学取舍。
- **工艺链规划**：理解从设计文件到物理原型的最优路径，平衡精度、速度与成本。
- **表面处理把控**：掌握从喷砂拉丝到功能涂层的质感控制参数。
- **制造经济性判断**：建立单件原型与小批量生产之间的成本意识。
- **跨学科整合**：将 CMF 决策与结构设计、电子功能、用户体验统一考虑。

## 🧱 2. 材料分类

### 2.1 工程塑料族

从外观壳体到耐磨功能件，建立课程中最常用的塑料材料选择逻辑。

| 项目 | 内容 |
|---|---|
| **代表材料** | ABS、PC、POM、PEEK、TPU、尼龙 |
| **关键词** | 塑料基材、强度/韧性、触感 |
| **执行路径** | 按刚性、韧性、耐温和耐磨建立材料表 → 区分展示件、结构件和运动件需求 → 选择匹配的打印、CNC 或注塑路径 → 记录材料成本、可得性和表面处理兼容性 |
| **输出目标** | 让学生先判断材料角色，再选择制造工艺 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/polymers/peek/) · [本地课程文档](/courses/fab-course/doc/0manage/material.html) |

### 2.2 金属材料族

围绕铝、铜、钛、不锈钢等金属基材，理解重量、强度、导热、导电和表面处理之间的取舍。

| 项目 | 内容 |
|---|---|
| **代表材料** | 铝合金、铜合金、钛合金、不锈钢 |
| **关键词** | 铝合金、铜合金、钛合金 |
| **执行路径** | 按重量、强度、散热和成本建立选择标准 → 匹配 CNC、钣金、SLM 或铸造路径 → 预留阳极、拉丝、喷砂或电镀余量 → 用真实样板比对颜色、反光和触感 |
| **输出目标** | 把金属 CMF 训练落实到结构和制造约束 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/metal/al/) · [本地课程文档](/courses/fab-course/doc/8CNC_manufacture/cnctype.html) |

### 2.3 纤维增强复合材料

以 PA-CF、PA-GF、ABS-GF 等复合材料建立轻量化原型中的强度方向、层纹、磨耗和加工风险判断。

| 项目 | 内容 |
|---|---|
| **代表材料** | PA-CF、PA-GF、ABS-GF、碳纤维纹理 |
| **关键词** | PA-CF、PA-GF、轻量化 |
| **执行路径** | 识别主受力方向与各向异性风险 → 选择连续纤维、短纤维或粉末烧结路径 → 处理喷嘴磨耗、收缩和表面粗糙度 → 通过夹具或实测验证刚度收益 |
| **输出目标** | 让复合材料不只停留在外观纹理，而成为结构决策工具 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/componsites/pacf/) · [本地课程文档](/courses/fab-course/doc/0manage/material.html) |

## 🖨️ 3. 增材制造工艺

### 3.1 FDM 熔融沉积

适合课程早期体量、装配、握持和机构干涉验证。重点在层纹方向、支撑策略、壁厚和嵌件预留。

| 项目 | 内容 |
|---|---|
| **适用材料** | PLA、ABS、ASA、TPU、PA-CF |
| **关键词** | 结构草模、低成本、装配验证 |
| **执行路径** | 按受力方向确定打印姿态 → 预留螺母、热熔铜柱与走线槽 → 用 0.2mm 层高完成首版验证 → 对关键面进行打磨、补土或喷漆 |
| **输出目标** | 快速得到可握持、可装配、可测试的第一轮样机 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/polymers/asa/) · [本地课程文档](/courses/fab-course/doc/3_3dprinter/2.FDM3Dprintingbackground.html) |

### 3.2 LCD / DLP 光固化

适合高精度外观件、小型结构、透明罩、CMF 质感样板。重点训练支撑痕迹、清洗固化和表面处理。

| 项目 | 内容 |
|---|---|
| **适用材料** | 标准树脂、透明树脂、韧性树脂 |
| **关键词** | 精细表面、透明树脂、视觉样件 |
| **执行路径** | 拆分可见面与支撑面 → 设置排液孔和空腔壁厚 → 清洗、二次固化并修整支撑痕迹 → 通过喷涂或抛光形成最终质感 |
| **输出目标** | 让设计意图在比例、边缘和表面细节上被准确表达 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/manufacture/add/lcddlp/) · [本地课程文档](/courses/fab-course/doc/3_3dprinter/5.SLAbackground.html) |

### 3.3 SLS / MJF 尼龙烧结

适合卡扣、铰链、轻量化壳体和耐冲击结构。用它连接外观样机与功能样机。

| 项目 | 内容 |
|---|---|
| **适用材料** | PA12、PA11、TPU、PA-CF |
| **关键词** | 功能样机、尼龙、小批量 |
| **执行路径** | 按功能面定义关键公差 → 为粉末清理预留孔位与开口 → 验证卡扣、铰链、耐冲击结构 → 染色、喷砂或浸渗提升外观一致性 |
| **输出目标** | 把 CMF 样件推进到接近真实使用场景的功能验证 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/polymers/pa/) · [本地课程文档](/courses/fab-course/doc/3_3dprinter/7.hpmjf.html) |

## 🔩 4. 减材制造工艺

### 4.1 CNC 精密加工

适合铝合金外壳、旋钮、支架、散热件和高精度装配面。关注刀路可达性、倒角、纹理方向和后处理余量。

| 项目 | 内容 |
|---|---|
| **适用材料** | 铝合金、铜合金、钛合金、POM、亚克力 |
| **关键词** | 铝合金、高精度、真实材质 |
| **执行路径** | 把外观面和基准面分离管理 → 约束内角半径和刀具可达性 → 预留阳极、喷砂或抛光余量 → 用装配件复核孔位和配合公差 |
| **输出目标** | 建立学生对真实材料、加工限制与工业质感之间关系的判断力 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/en/prototype/manufacture/substractive/cnc/) · [本地课程文档](/courses/fab-course/doc/8CNC_manufacture/cnctype.html) |

## ⚙️ 5. 成型工艺

### 5.1 钣金折弯

适合底盘、支架、防护罩、电控盒和课程机器人框架。关键是折弯半径、展开图、孔边距和表面喷涂策略。

| 项目 | 内容 |
|---|---|
| **适用材料** | 冷轧板、镀锌板、铝板、不锈钢 |
| **关键词** | 支架、外壳、展开图 |
| **执行路径** | 用折弯线定义结构逻辑 → 检查孔边距和干涉风险 → 输出展开图与折弯顺序 → 结合喷粉、拉丝或阳极完成 CMF |
| **输出目标** | 让原型从塑料打印件升级为更接近工业装配的结构件 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/metal/al/) · [本地课程文档](/courses/fab-course/doc/2cad/fusion360-sheetmetal.html) |

### 5.2 注塑与硅胶覆膜

适合讲解拔模、分型线、缩水、嵌件与软胶包覆。课程不只做样机，还要理解未来量产中的设计取舍。

| 项目 | 内容 |
|---|---|
| **适用材料** | ABS、PC、PP、TPU、硅胶、PU |
| **关键词** | 量产判断、拔模、软硬复合 |
| **执行路径** | 标注拔模方向与分型线 → 检查壁厚、筋位和缩水风险 → 用软胶区域定义触感与防滑性能 → 建立样机成本与模具成本对照 |
| **输出目标** | 把 CMF 从视觉选择推进到结构、触感和制造经济性的综合决策 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/material/polymers/tpu/) · [本地课程文档](/courses/fab-course/doc/9MOLD/mold.html) |

## 🎨 6. 表面处理

### 6.1 喷砂、抛光与拉丝

统一管理表面粗糙度、反光、边缘倒角和触感。适合建立 CMF 样板册，让同一造型呈现多种质感。

| 项目 | 内容 |
|---|---|
| **适用材料** | 铝合金、不锈钢、树脂、亚克力 |
| **关键词** | 质感控制、样板册、边缘处理 |
| **执行路径** | 记录基材、砂号、压力和处理时长 → 控制可见边的倒角一致性 → 对比哑光、半哑光与镜面效果 → 将样板编号并归档到课程材料库 |
| **输出目标** | 让学生能用参数描述质感，而不是只说"高级"或"好看" |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/postprocess/postprocess/) · [本地课程文档](/courses/fab-course/doc/3_3dprinter/8.postprocess.html) |

### 6.2 导电与功能涂层

适合把外观件、电磁屏蔽、触摸电极和结构件连接起来。可作为材料、电子和产品体验的跨学科案例。

| 项目 | 内容 |
|---|---|
| **适用材料** | 导电漆、铜箔、树脂件、塑料壳体 |
| **关键词** | 功能表面、触摸电极、屏蔽 |
| **执行路径** | 区分装饰层与功能层 → 规划接地、触摸或屏蔽路径 → 用遮蔽和分区控制涂层边界 → 完成导通、附着力和耐磨测试 |
| **输出目标** | 把 CMF 训练从外观审美扩展到功能材料应用 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/postprocess/functional/conductive/) · [本地课程文档](/courses/fab-course/doc/4electric_design/basicknowledge.html) |

## 🔄 7. 工作流集成

### 7.1 数字沙盘原型

面向课程展示、展厅模型和产品路演的综合原型路径，强调模型比例、模块拆分、灯光/交互与可维护结构。

| 项目 | 内容 |
|---|---|
| **适用材料** | 亚克力、树脂、铝型材、灯带 |
| **关键词** | 展示模型、结构拆分、交互陈列 |
| **执行路径** | 定义展示场景与观察距离 → 拆分外壳、底座、灯光与控制模块 → 建立比例模型并完成装配校验 → 输出维护清单与展示说明 |
| **输出目标** | 形成可巡展的高完成度原型，而不是单次拍摄用样机 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/sandtable/) · [本地课程文档](/courses/fab-course/doc/1projectmanage/Assessment1.html) |

### 7.2 探索数字制造

将 CAD、材料选择、工艺约束、后处理与复盘文档串联成一个可重复的课程流程，适合 CMF 小组项目管理。

| 项目 | 内容 |
|---|---|
| **适用材料** | PLA、ABS、亚克力、木材 |
| **关键词** | FabLab、课程流程、复盘文档 |
| **执行路径** | 建立需求与造型边界 → 选择低风险快速工艺完成第一版 → 通过 CMF 样板校正颜色、质感与结构 → 沉淀 BOM、工艺参数与失败样本 |
| **输出目标** | 让学生把工具使用转化成可迁移的工程决策能力 |
| **参考文档** | [ProFabX 源页面](https://profabx.com/prototype/exploredigitalfabrication/) · [本地课程文档](/courses/fab-course/) |
