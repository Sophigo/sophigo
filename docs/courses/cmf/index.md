# CMF 设计美学与规范 (Color, Material, Finish)

CMF 是定义工业产品最终质感、档次与用户触觉体验的核心规范。本指南主要阐述 Sophigo 移动机器人底盘及控制器外壳的金属加工与表面处理标准。

## 1. 铝合金表面处理

铝合金外壳具备轻量化与优异的散热性能。在机械加工（CNC）完成后，需经历喷砂与化学处理：

- **喷砂粒度标准 (Sandblasting):**
  - #120 号金刚砂: 质感粗犷，防滑耐磨，适用于户外机器人底盘侧板。
  - #180 号金刚砂: 质感细腻，触感柔和，适用于手持遥控器及上盖。

- **阳极氧化工艺 (Anodizing):**
  > [!IMPORTANT]
  > 通过电化学方法，在铝合金表面生成一层致密且可染色的氧化铝保护膜。
  > 我们的标准**阳极氧化工艺**要求膜厚达到 8μm - 12μm，以确保色彩均匀性与抗腐蚀寿命。
  > 推荐色值包含：太空灰、极客黑与**克莱因蓝 (#002FA7)**。

---

## 2. 涂装工艺参数对比

| 工艺名称 | 表面硬度 | 耐磨性 | 生产成本 | 环保等级 | 适用部位 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **阳极氧化** | 300-400 HV | 极佳 | 中等 | 高 | 铝合金外壳 |
| **粉末喷涂** | 2-3 H (铅笔) | 良好 | 较低 | 中等 | 钢制防护栏 |
| **手感漆涂装** | < 1 H | 一般 | 较高 | 中等 | 遥控器手柄 |
