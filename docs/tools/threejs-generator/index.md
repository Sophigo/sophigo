# 3D生成器

ProFabX / Sophigo 的在线 3D 预览与报价分析工具，支持多格式导入、装配关系编辑、壁厚风险判断和制造报价估算。

</p>

## 功能介绍

- **多格式载入**：支持 STL / OBJ / STP / STEP / IGES / URDF / F3D / F3Z。
- **零件与装配体预览**：单零件、装配体和批量导入均可进入 3D 视图，多零件使用 Google 色系区分。
- **报价分析**：保留材质、密度、工艺、后处理、重量、费用统计和 Excel 导出。
- **壁厚分析**：提供阈值滑条和红到蓝风险条，用于初步识别制造风险。
- **装配关系编辑**：可新增、选中、更新和删除零件之间的固定、转动、直线、平面、球铰关系。
- **深色 / 浅色模式**：默认浅色，可在工具右上角切换。

## 使用步骤

1. 点击上方链接打开工具页面。
2. 在左侧选择文件、材质、密度、工艺和后处理。
3. 在中间 3D 视口选择零件并编辑装配位置。
4. 在右侧切换“报价分析 / 壁厚分析 / 装配编辑”分页。
5. 完成后导出 Excel 或生成本地分析快照。

> [!NOTE]
> STL / OBJ 当前可在前端直接解析；URDF 当前解析结构预览；STP / STEP / IGES / F3D / F3Z 会先进入 CAD 待转换状态，后续可接 OpenCascade.js 或服务端 CAD 转换。
