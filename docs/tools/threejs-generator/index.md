# Three.js 3D 生成器

Sophigo 提供的在线 3D 渲染与场景生成工具，帮助软硬件开发者快速可视化机器人模型与机械结构。

## 功能介绍

- **参数化建模**：输入长度、宽度、轮毂尺寸等核心参数，一键生成对应的底盘 3D 模型。
- **实时粒子渲染**：基于 Three.js 实现低功耗、高性能的高画质 3D 效果。
- **模型导出**：支持导出标准的 STL/OBJ/GLTF 格式，无缝接入快速原型数字制造（如 3D 打印）。

## 使用步骤

1. 在左侧面板配置您的移动底盘规格或零件属性。
2. 实时观察右侧 3D 视口中渲染的物理质感。
3. 点击“导出 GLTF”或者“下载 G-code”。
