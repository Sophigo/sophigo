# AI 视频生成工具

利用先进的多模态生成模型，输入产品 3D 渲染图或简单的文字描述，即可自动合成出极具质感、动态流畅的工业产品展示视频与 CMF 宣传片。

## 典型应用场景

- **创意产品概念展示**：在还没有制造出实体样机前，用 AI 视频快速展示产品运动和工作状态。
- **CMF 效果动态模拟**：模拟阳光和光源移动下，金属表面阳极氧化和精细喷砂的微观反光质感。
- **教学视频自动生成**：结合语音与三维图纸，一键生成移动机器人的组装与调试教程视频。

## 使用建议

1. 尽量上传背景纯净的白模或者高质量的 STL/GLTF 文件作为画面底图。
2. 在 Prompt 中明确指定光影（如：软调漫反射、克莱因蓝主调）和运动相机轨迹（如：环绕、推拉）。
