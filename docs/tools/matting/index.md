# AI 抠图与产品图像生成

本工具基于深度学习算法（如 Segment Anything），实现发丝级的高精度发丝与工业零件主体背景分离。一键去除杂乱的环境，并可根据 CMF 规范一键替换为高级纯色或极简工业渲染背景。

## 核心功能

- **极速背景分离**：秒级处理，自动识别并精确勾勒产品本体。
- **CMF 背景模版匹配**：预设契合 SophiGo 克莱因蓝、太空灰及极客黑等工业质感的高级灰渐变背景。
- **多角度阴影自动生成**：根据边缘曲线，智能生成附着物理属性的阴影效果，使抠出的产品图更显真实与高级。

## 使用流程

1. 拖入您的物理样品照片或渲染图。
2. 算法将自动识别出产品主体（可点击微调）。
3. 选择所需的背景模式与材质规格，点击下载高分辨率 PNG 图。
