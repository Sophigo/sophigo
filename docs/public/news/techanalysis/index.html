Physical Computational Engineering

Leap71

Leap71 致力于推动"计算工程"范式。通过编写算法代码（C#）表达物理物体的"几何与逻辑 DNA"，完美适配高精度金属 3D 打印，可在几分钟内自主生成航空级喷气引擎或高能热交换器。

  • 代码即设计 DNA：物理方程式、热力学参数和制造约束编写为代码
  • PicoGK 体素内核：高精度体素技术，支撑无限复杂的网格格栅结构
  • Noyron CEM：确定性物理驱动的大计算工程模型，非神经网络
  • 无缝增材制造：直接生成金属打印切片路径及 G-code
Agentic Articulated 3D Generation

Articraft

开源"智能体 3D 关节资产生成系统"，通过"代理—沙箱执行—验证反馈"闭环迭代，让 AI 自动编写 Python CAD 脚本，创造出具备物理关节、碰撞体积且能够移动的"活资产"。

  • LLM 智能体闭环：AI 编写脚本并在受限 Harness 内执行与自我修正
  • CadQuery 参数化：LLM 友好的 Python 接口进行语义级构筑
  • 仿真就绪关节：自动解算铰链极限角度与位移，输出 URDF
  • Articraft-10K：超 10,000 个带真实物理运动属性的 3D 模型

技术架构与壁垒剖析

技术方案对比

两条截然不同的算法与几何构筑路径

尽管两者均倡导"代码生成 3D 资产"的理念，底层引擎与架构逻辑服务于不同的终极目标：

Leap71: 物理密度与体素流
采用 PicoGK 体素引擎（窄带有符号距离场 SDF），将三维空间表示为海量网格标量场。Noyron 是确定性引擎，基于第一性原理物理，非统计概率模型。输出直接适配增材制造。
Articraft: 智能体沙箱与运动解算
基于 LLM Agent 编写 CadQuery 脚本。LLM 编写高级几何组合逻辑，通过受限沙箱编译器进行几何正确性、重叠度、关节自由度的验证与微调，最终合成物理模型（URDF）。
Leap71 核心竞争力

击碎传统 CAD 复杂度天花板

传统 CAD 基于 B-Rep，设计含数百万微小散热胞腔时崩溃。Leap71 采用体素核心 + 窄带 SDF，使计算速度与复杂度脱钩。Noyron 编码了工程启发式规则和物理约束，输出的是生产就绪设计——无需人工清理后处理。

Articraft 核心竞争力

赋予 3D 模型真实的"运动灵魂"

当今 3D 生成 AI 产出的是"静止雕像"。Articraft 生成可活动的装配体（可拉开的抽屉、能回弹的剪刀），自动编写 URDF 关节逻辑，让机器人在 Isaac Gym 或 MuJoCo 中立刻操作。比通用编码代理和先前最先进方法生成更高质量资产。
